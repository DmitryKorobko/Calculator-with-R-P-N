SteinCalc

Description:

The dual-mode calculator. The possibility of calculating both the classic calculator, as well as the ability to calculate complex expressions using reverse polish notation. To activate / deactivate the use of RPN, you must press the appropriate button - "RPN". When using classic mode all buttons except the brackets. When using RPN, some buttons are disabled. Among the additional features implemented include: memory operations calculator, such as recording in memory, reading from memory, delete from memory, adding to the number in memory and subtracts from the number in memory, also realized the calculation of the square root of a number raised to the square to cube and in any degree, added the numbers "Pi" and "e", the calculation of the sine and cosine operation with interest, the ability to change the +/- sign, as well as removing the last character. The buttons can be pressed using the keyboard or mouse. Additional keys: "P" - the number Pi, "e" - exponent, "C" - discharge of all values on the screen, "Backspace" - delete the last character, "Enter" - operation "equal", "R" - activation mode reverse Polish Notation, "%" - the operations with percents.
